## KJV Bible
A King James Bible website made with PHP.

### Dependencies:
1. PHP
2. SQLite3

### How to run:
1. Navigate to the project directory:
   ```bash
   cd kjv-bible
   ```
2. Start the PHP server:
   ```bash
   php -S localhost:8000
   ```
3. Open your browser and go to [localhost:8000](http://localhost:8000)
